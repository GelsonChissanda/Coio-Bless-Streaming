# Coio Bless Streaming
Plataforma Oficial de Streaming Musical  Site profissional desenvolvido para um único artista, permitindo que fãs ouçam, explorem e baixem músicas gratuitamente. A plataforma organiza o catálogo em álbuns, EPs, singles e faixas soltas, oferecendo uma experiência moderna inspirada no Spotify.  Recursos Reprodução de músicas online Download gratuito de faixas Letras sincronizadas estilo Spotify Biografia completa do artista Discografia organizada Contadores de reproduções e downloads Painel administrativo completo Upload de músicas e capas SEO otimizado Design responsivo para mobile e desktop Tecnologias Next.js 15 TypeScript Tailwind CSS Supabase Cloudinary Shadcn/UI Vercel  Projeto criado para servir como a presença digital oficial do artista, centralizando sua música, história e lançamentos em uma única plataforma moderna e escalável.
